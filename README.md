### Log Analyzer (Electron) üîß

[LogAnalyzer](https://github.com/hwanjongyu/LogAnalyzer) is a high-performance desktop log analysis application.

- **Tech Stack:** Electron, React, TypeScript, Tailwind CSS, Zustand, React Virtuoso.
- **Key Features:**
  - **High Performance:** Smoothly handles 100,000+ lines with virtualized rendering.
  - **Advanced Filtering:** Regular expression support with Highlight, Include, and Exclude filters.
  - **Filter Management:** Save/load filter configurations as JSON and manage custom filter tabs.
  - **Smart View:** "Show Only Highlighted" mode for focused analysis.

### Log Analysis GUI (Python) üêç

[loganalysis_gui](https://github.com/hwanjongyu/loganalysis_gui) is a Python-based log analysis tool with a focus on real-time monitoring.

- **Tech Stack:** Python 3.8+, PyQt5, ADB (Android Debug Bridge).
- **Key Features:**
  - **ADB Integration:** Real-time log streaming from Android devices via `adb logcat`.
  - **High-Performance Architecture:** Model-View architecture handling millions of lines.
  - **Persistent Profiles:** Customizable filter profiles stored in JSON.
  - **Multi-layered Filtering:** Comprehensive Include/Exclude/Highlight system with regex support.

### MediaHub üçø

[MediaHub](https://github.com/hwanjongyu/MediaHub) is a high-performance, local-first media management tool.

- **Tech Stack:** Rust (Tauri v2), React, TypeScript, SQLite3 (sqlx), Tokio, Rayon.
- **Key Features:**
  - **Ultra-Fast Indexing:** Uses Rust and parallel processing to scan thousands of files per second.
  - **Memory Efficient:** Streaming architecture keeps RAM usage low even with 100,000+ files.
  - **Virtualized Grid:** Smooth browsing experience with `react-window`.
  - **Secure Local Streaming:** Custom protocol for secure local file access.

### PrivacyLens üîç

[PrivacyLens](https://github.com/hwanjongyu/PrivacyLens) is a privacy-first mobile camera application designed to redact sensitive information in real-time.

- **Tech Stack:** Kotlin, Jetpack Compose, MediaPipe, LiteRT (TensorFlow Lite), ML Kit, CameraX.
- **Key Features:**
  - **Real-time Redaction:** On-device AI for face blurring (Gaussian, Pixelate, Emoji).
  - **PII Sentinel:** Automated detection and redaction of IDs, financial data, and personal info.
  - **On-device AI:** Powered by Gemini Nano via AICore for proactive security audits.
  - **Privacy-First:** All processing happens locally; data never leaves the device.
